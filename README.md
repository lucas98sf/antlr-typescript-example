# Compiler to TypeScript using ANTLR
![image](https://github.com/user-attachments/assets/2651ad10-d986-4f4a-9e5d-b4ee91b368b5)

## To install dependencies:

```bash
bun install
```

## To run tests:

```bash
bun test
```

## To run the compiler

```bash
bun start
```

Opens the compiler at http://localhost:5173/
